  • 🆕 Máster Especialista en Series Temporales (Forecasting) con Python

    • Modelado, diagnóstico y predicción de series temporales desde cero hasta nivel experto.

    • Recomendado tras: Machine Learning con Python.

  • 🆕 Máster Especialista de Deep Learning en Python con PyTorch

    • Redes neuronales profundas: diseño, implementación y evaluación desde 0 a experto.

    • Recomendado tras: Machine Learning con Python.

  • Máster Especialista en Optimización de Redes Neuronales con Python y TensorFlow

    • Curso avanzado para desarrollar y optimizar redes profundas con TensorFlow.

    • Recomendado tras: Deep Learning con Python y Keras y/o Máster en Ciencia de Datos con Python.

  • Máster en Procesamiento del Lenguaje Natural (NLP) con Python

    • Modelos clásicos, secuencias y Transformers con TensorFlow.

    • Recomendado tras: Deep Learning con Python y Keras.

  • Máster Especialista en Machine Learning Ensemble con Python

    • Modelos ensemblados: Bagging, Boosting y Stacking.

    • Recomendado tras: Máster en Procesamiento del Lenguaje Natural con Python.

  • Máster Especialista en Ciencia de Datos con Python

    • Machine Learning y Deep Learning aplicado: de básico a experto.

    • Incluye fundamentos esenciales de Machine Learning y Deep Learning.

  • Deep Learning con Python y Keras (Redes Neuronales)

    • Diseño y entrenamiento de redes neuronales desde básico a experto.

    • Recomendado tras: Machine Learning con Python (Avanzado).

  • Machine Learning con Python (Aprendizaje Automático)

    • Supervisado, no supervisado y redes neuronales artificiales.
